# 拳魂 SOUL FIST

KOF 风格的像素格斗游戏(HTML5 Canvas,零依赖)。人机对战,2 个可选角色,完整格斗机制。

## 运行

```bash
cd soul-fist
python3 -m http.server 8787
# 打开 http://localhost:8787
```

(直接双击 index.html 用 file:// 打开也能玩。)

## 键位

| 键 | 功能 |
|---|---|
| A / D | 左右移动(空中可漂移转向);所有键位可自由组合 |
| W | 跳跃(按住方向 = 带方向起跳;冲刺/后撤中也可跳) |
| 按住后方向 | **格挡(KOF 式方向防御)**:命中一刻须按住远离敌人的方向;出招/跳跃/冲刺/下蹲中防不了;有护条,被打满会**破防**;绕背可破 |
| S | **下蹲**:蹲 J = 低位削足;蹲 K = 斜上升斩(挑空对空,隼人「蹲升·月轮」/剑二「蹲撩·逆风」) |
| J | 轻击(地面/空中可用,连按来回变招) |
| K | 重击(连按变招);**空中按 = 下坠重砸(落地冲击波)** |
| U | 必杀技(独立冷却) |
| I | 超必杀(气槽满 100 才能放,带演出) |
| 双击 D | 向前冲刺(可取消进攻击) |
| 双击 A | 后撤闪避(起手无敌帧) |
| Esc / P | 暂停 · M 静音 · H 隐藏战斗提示 |
| T / R | 练习场专用:T 切换木桩模式(站桩/自动格挡/AI 反击),R 复位 |

**连招**(全部经端到端实测):起手 `J → J → K`,收尾三选一:
- `K`(第二段重击 = 前踏终结斩,**击倒**)—— 隼人 35 / 剑二 29 伤害
- `U`(隼人限定,月牙疾斩收尾,击倒)—— 35 伤害
- `I`(超必杀演出收尾,需气满)—— 隼人 47 / 剑二 45 伤害

剑二的 `U` 是远程手里剑,用于立回牵制,不参与连招。
连按 J / K 会**来回变招**(高斩/低扫、下劈/撩斩交替,刀路带倾角);
作为第 3 hit 及以后命中的连锁招享受 **BONUS 伤害 ×1.3**(金色提示)。
连击整体有伤害衰减(3 hit 后 72%,5 hit 后 50%)。冲刺/后撤中可随时起跳(带冲刺动量)。

## 角色

- **隼人 HAYATO**(力量型武士):重剑大范围,必杀「月牙·疾斩」突进斩,超必杀「无月·连狱斩」。
- **剑二 KENJI**(速度型忍者):移速快,必杀「影·手里剑」远程飞镖,超必杀「残影·灭杀阵」瞬身背刺连段。

## 机制清单

移动/跳跃/冲刺/后撤闪避(i-frames)、动量继承(移动/冲刺中攻击带滑步,跳跃攻击带弧线)、
空中轻击+下坠重砸(俯冲姿态/落地冲击波/AoE 判定)、
格挡+chip 伤害+防不死、**护条系统(格挡积累护伤,满 100 破防大硬直;必杀/超必杀猛削护条)**、
**方向格挡(格挡时朝向固定,瞬身绕背可破)**、轻重攻击、
链式取消连招+伤害衰减+连击计数、必杀技冷却、气槽(打击/受击/格挡攒气)、
超必杀暗屏演出+多段锁定连段、击倒/倒地/起身无敌、hitstop/屏震/慢镜头 KO、
回合制 BO3(60 秒计时、TIME UP 判定、PERFECT/DOUBLE K.O.)、
AI 三档难度(反应/防御/闪避/连招跟进,护条告急会主动闪避)、选人+难度选择+键位教学界面、
**练习场模式(木桩三态:站桩/自动格挡/AI 反击,T 切换;R 复位;气槽无限;脱战自动回血;无计时无 KO)**、
180ms 输入缓冲(120Hz+ 高刷屏不吃键,连招输入更宽容)。

音频全部为 WebAudio 实时合成(打击感音效 + 芯片音乐 BGM),无音频素材文件。

## 调试参数

- `?fight=1&p1=mack&p2=kenji&ai=hard` 直接进战斗
- `&demo=1` 双 AI 观战模式
- `&debug=1` 显示碰撞框
