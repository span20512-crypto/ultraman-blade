# 刀魂 SOUL BLADE

KOF 风格的和风像素格斗游戏 —— 纯 vanilla JS + Canvas,**零依赖、零构建**,首屏 ~12MB。
人机对战,8 个可选英雄,完整格斗机制(连锁/方向格挡/护条破防/浮空连段/超必演出)+ 练习场 + 招式图鉴。新增「光之对决」本地双人模式,可各自选角对战。

🎮 在线试玩:https://soul-blade.pages.dev (需要电脑 + 键盘操作)

## 运行

```bash
python3 serve.py          # http://localhost:8787(开发用,带 no-store 头)
```

或任意静态服务器伺服本目录。

## 键位

| 键 | 功能 |
|---|---|
| A / D | 左右移动(空中可漂移转向) |
| W | 跳跃(带方向起跳;冲刺/后撤中也可跳) |
| 按住后方向 | **格挡(KOF 式方向防御)**:命中一刻须按住远离敌人的方向;出招/跳/冲刺/蹲中防不了;格挡积累**护条**,满 100 破防大硬直;绕背可破 |
| S | 下蹲:蹲 J = 低位削足(可连打);蹲 K = 挑空升斩(可接超必) |
| J / K | 轻击 / 重击(连按来回变招;空中 K = 下坠重砸) |
| U | 必杀技(独立冷却):剣二 = 突进斩,隼人 = 手裏剣(命中确认可接超必) |
| I | 超必杀(气槽满 100,带演出) |
| 双击 D / A | 前冲刺(可取消进攻击)/ 后撤闪避(起手无敌帧) |
| Esc / P · M · H | 暂停 · 静音 · 隐藏提示 |
| T / R | 练习场:木桩三态切换 / 复位 |

### 光之对决（本地双人）

标题菜单在「修行」下方选择 **ULTRA DUEL · 光之对决**。两名玩家可分别使用两把键盘（浏览器会合并键盘输入，请使用下方两组不同键位）。

| 玩家 | 选角 / 移动 | 跳跃 | 近战 | 光线 |
|---|---|---|---|---|
| 1P | W / A / S / D | W | J | U |
| 2P | 方向键 | ↑ | 小键盘 1 | 小键盘 4 |

选角池共 **16 名战士**（4×4 网格）：8 位奥特曼英雄（初代、赛文、赛罗、泰罗、迪迦、戴拿、盖亚、泽塔）+ 8 只怪兽（怪兽一号/二号、巴尔坦星人、哥莫拉、金古乔、雷德王、五帝王、玛迦大蛇）。英雄可打怪兽、怪兽可打怪兽，随意组合。光之对决包含 60 秒计时、生命值、近战击退、可消耗能量的远程光线、胜负结算与快速重开。

标题菜单选 HOW TO PLAY(或暂停菜单按 K)可打开**招式图鉴**:左侧全招式表,右侧真引擎实时演示。

![招式图鉴](docs/howto.webp)

## 角色

8 位奥特曼英雄,分两套操作框架(帧数据/连招/平衡完全同基线,只换主题色与立绘):

- **力量系**(重击大伤害,一套连段定胜负):**ULTRAMAN 初代**(红)· **TARO 泰罗**(橙)· **GAIA 盖亚**(绯红)。
- **速度系**(移速手长占优,飞行道具立回,瞬身背刺超必):**SEVEN 赛文**(翠绿)· **ZERO 赛罗**(紫)· **TIGA 迪迦**(魅紫)· **DYNA 戴拿**(青)· **Z 泽塔**(蓝)。

对手侧(CPU)为怪兽阵营,选人页右侧实时预告下一位对手。

**基础连招**:`J → J → K` 起手,收尾三选一:`K`(击倒终结,剣二 31 / 隼人 30)、`U`(剣二限定,32)、`I`(超必收尾,双方 46)。蹲 K 挑空后可接空中追击(上限 1 次)或直接接超必。3 hit 及以后的连锁招享 ×1.3 终结加成,整体带伤害衰减。

## 机制清单

方向格挡 + chip 伤害 + 护条破防 · 链式取消连招 + 伤害衰减 · 浮空连段(空中追击上限 1)·
必杀冷却 + 气槽(打击/受击/格挡攒气)· 超必暗屏演出 · 击倒/起身无敌 · hitstop/屏震/慢镜 KO ·
回合制 BO3(60 秒计时 / TIME UP / PERFECT / DOUBLE K.O.)· AI 三档难度(困难 = 读指令的 SNK Boss 式鬼级)·
练习场(木桩三态 / 无限气 / 自动回血)· 180ms 输入缓冲(高刷屏不吃键)。

音频:音效为 WebAudio 实时合成;全程 BGM 为授权英雄摇滚主题(跨屏无缝续播、无缝循环);AI 生成的和风器乐保留在 assets/audio/bgm/ 作候补。
